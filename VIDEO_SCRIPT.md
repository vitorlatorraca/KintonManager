## Recording Tips

### Screen Recording Setup
1. **Resolution**: Record at 1920x1080 (Full HD) minimum
2. **Frame Rate**: 30fps or 60fps for smooth animations
3. **Browser**: Use Chrome or Firefox with clean browser (no extensions visible)
4. **Zoom**: Set browser zoom to 100%
5. **Cursor**: Use a clean cursor, or hide it during transitions

### Tools Recommended
- **Screen Recording**: 
  - OBS Studio (free, open source)
  - Loom (easy to use)
  - QuickTime (Mac)
  - Windows Game Bar (Windows)
- **Editing**:
  - DaVinci Resolve (free, professional)
  - Adobe Premiere Pro
  - Final Cut Pro (Mac)
  - CapCut (free, mobile-friendly)

### Recording Checklist
- [ ] Clear browser cache and close unnecessary tabs
- [ ] Use demo accounts (create test customer and manager accounts)
- [ ] Prepare sample data (a few customers with different stamp counts)
- [ ] Test all flows before recording
- [ ] Record in a quiet environment
- [ ] Use good lighting if including face cam
- [ ] Record audio separately if possible (better quality)

## Quick Recording Workflow

1. **Prepare** (15 min)
   - Set up demo accounts
   - Clear browser
   - Test all flows

2. **Record** (30-45 min)
   - Record each section separately
   - Do multiple takes if needed
   - Keep raw footage organized

3. **Edit** (2-3 hours)
   - Import footage
   - Cut and arrange scenes
   - Add narration/music
   - Add text overlays
   - Color correction

4. **Review** (30 min)
   - Watch full video
   - Check audio levels
   - Verify all information is correct
   - Get feedback if possible

5. **Export & Upload** (30 min)
   - Export in high quality
   - Upload to platforms
   - Add descriptions and tags

## Alternative: Simple Screen Recording Approach

If you want a quicker approach:

1. **Record a live demo** (10-15 min)
   - Start from homepage
   - Show customer login and features
   - Show manager login and dashboard
   - Narrate as you go

2. **Add simple edits** (30 min)
   - Trim unnecessary parts
   - Add title card at start
   - Add end card with links
   - Basic transitions

3. **Export and share**

This approach is faster but less polished. The detailed script above will give you a more professional result.
